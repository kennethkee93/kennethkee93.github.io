FAQ
Q: Who created this web app? This web app was developed by members of the Emerald Hills EV community to support efficient scheduling and communication regarding EV charger reservations.

Q: Is this web app affiliated with Emerald Hills Management? No, this web app is independent and is not affiliated with Emerald Hills Management in any way.

Q: What is the purpose of this web app? The primary goal of this web app is to facilitate better communication and planning among EV owners regarding charger availability.

Q: How do I cancel a reservation? To cancel a reservation, please use the unique cancellation link provided in your booking confirmation email.

Q: How do I modify my reservation? To make changes to an existing reservation, please cancel your current booking using the cancellation link, then submit a new reservation.
